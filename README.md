# ACS-Tweaks

A tweak mod for Amazing Cultivation Simulator.

## Features

- [x] Tweaks some grindy/slow aspects of the game like the Mental State (Mind), Practice Skill Exp Rate, Extreme Volatility of everything during GC breakthroughs.
- [x] Changes the cost of Skill Learning (Easy).
- [x] Increases Base Move Speed of Disciples.
- [x] Makes slow events faster.
- [x] Qi Gathering stabilization completes in 1 day instead of 5 (make sure you edit your other mods to display it correctly).
- [x] Makes Law Matches more realistic (Easy?).
- [ ] Other Upcoming Easy-Mode tweaks to normalize the game and remove it's exponential boredom.

## Append

Be careful when changing the name of the project or the assembly, _ModloaderLite_ will not be able to find it.
